Edu-Smart (데모 웹앱)
====================

시작
----
1. index.html 포털에서 로그인 후 역할에 맞는 메뉴로 이동합니다.
   - members.html — 관리자(admin)만 회원 등록·수정·삭제
   - employee.html — 신청자 (한도·전액 조회, 신청, Tesseract OCR 검수)
   - manager.html — 부서장 (1차 결재만)
   - admin.html — 교육담당자 (최종 검수·CSV)

2. 교육비·세미나는 수료증이 필수이며, 수료증 OCR으로 수료·이수(또는 수료증·
   이수증 등) 문구가 확인되어야 제출됩니다.

3. 신청 시 영수증/수료증 이미지에서 실제 OCR(kor+eng)으로 글자를 읽어,
   입력한 업체명·일자·금액과 비교합니다. 첫 실행 시 언어 데이터 다운로드로
   수십 초~수분 걸릴 수 있습니다(인터넷 필요).

4. index.html 의 "데모용 샘플 신청 넣기"로 결재 대기 건을 바로 넣을 수 있습니다.

데이터
------
- localStorage 키: edu-smart-v1 (세션·회원 목록·신청 데이터)
- 기본 계정: employee / manager / admin — 비밀번호 demo123 (평문 저장 데모)
- 증빙 이미지는 브라우저에만 저장됩니다.
